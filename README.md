# Intelligent Mess Management System

## Overview
A comprehensive mess management solution for hostels and residential facilities that handles meal planning, attendance, and billing.

## Features
- Menu planning and management
- Attendance tracking
- Billing system
- Complaint management
- Feedback collection
- Dietary preferences

## Tech Stack
- HTML5
- CSS3
- JavaScript
- Backend technologies
